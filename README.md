# p1-pong
Its the first part of the warm-up project series. Pong

## Features
- PvAI gameplay with beatable AI opponent
- State machine (Menu, Serve, Playing, Score, GameOver)
- Event-driven architecture (Observer pattern)
- Score system with with condition
- Audio feedback

## What I Learned
- Unity physics with Rigidbody2D
- Coroutines for timed sequences
- Singleton pattern for GameManager and AudioManager
- Separating concernes across Manager scripts
